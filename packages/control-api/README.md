# @claudexor/control-api

Internal package of [Claudexor](https://github.com/razzant/claudexor) — Loopback HTTP+SSE control surface for GUI/service clients: token-auth, Host/Origin guard, live run event stream with Last-Event-ID replay. Runner injected (no second scheduler).

Published as part of the Claudexor toolchain; it follows the monorepo's
lockstep version and has no separate semver contract. Use the `claudexor`
CLI (or `@claudexor/cli`) as the supported entry point.
